Traducción del [[_MOC Modelo de datos|modelo de datos]] a un esquema físico/lógico.

## Por definir
- Motor(es) y justificación (ADR).
- Esquema lógico (tablas/colecciones, relaciones).
- Estrategia para **datos sensibles** (cifrado, particionado, retención, auditoría).
- Índices y rendimiento.
